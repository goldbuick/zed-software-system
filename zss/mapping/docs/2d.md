# 2d.ts

**Purpose**: 2D coordinate utilities — index ↔ PT conversion, distance, rect/line enumeration. Uses PT from words/types.

## Dependencies

- `zss/words/types` — PT

## Exports

| Export | Description |
|--------|-------------|
| `indextox(index, width)` | `index % width` |
| `indextoy(index, width)` | `floor(index / width)` |
| `indextopt(index, width)` | `{ x, y }` from index |
| `pttoindex(pt, width)` | `pt.x + pt.y * width` |
| `ptdist(pt1, pt2)` | Euclidean distance |
| `ptstoarea(p1, p2)` | `"x1,y1,x2,y2"` string |
| `ptwithin(x, y, top, right, bottom, left)` | Point in rect |
| `rectpoints(x0, y0, x1, y1)` | All PTs in inclusive rect |
| `linepoints(x0, y0, x1, y1)` | Bresenham line PTs |
