## keyboard input

- arrow keys - $greenmove
- shift + arrow keys - $greenshoot
- enter - $greenok / accept
- escape - $greencancel / close
- tab - $greenmenu / layout

## controller input

- left stick - $greenmove
- right stick - $greenshoot
- left trigger - $greenalt
- right trigger - $green$meta
- left shoulder - $greencancel / close
- right shoulder - $greenok / accept
- d-pad - $greenmove
- a X - $greenshoot down
- b $9 - $greenshoot right
- x $254 - $greenshoot left
- y $30 - $greenshoot up
- menu - $greenmenu / change layout
