# zss firmware

firmware is how zss describes what #commands, and flags your code can use.

different codepages get different sets of firmware loaded:
  * board
    * all
    * audio
    * gadget
  * object
    * all
    * audio
    * object
    * gadget
  * terrain
    * all
  * charset
    * all
  * palette
    * all
